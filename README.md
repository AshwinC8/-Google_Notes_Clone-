## Setup
- To get a local copy of the code, clone it using git :
```
git clone https://github.com/AshwinC8/Google_Notes_Clone.git
cd Google_Notes_Clone
```

- Install Dependencies
```
npm install
```

- Now, you can start a local web server by running :
```
npm run dev
```
